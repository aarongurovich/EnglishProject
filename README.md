# EnglishProject

This repository contains the source code for **EnglishProject**, a multimedia web presentation comparing Artificial Intelligence and Human creativity. Built with standard HTML, CSS, and JavaScript, the project showcases various media examples—including generated images and composed music—side-by-side with their human counterparts to explore the differences in artistic expression.

## Getting Started

To view this project locally:

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/aarongurovich/englishproject.git](https://github.com/aarongurovich/englishproject.git)
    cd englishproject
    ```

2.  **Open the Application**
    Simply open the `index.html` file in your preferred web browser.
    * On Windows, double-click `index.html`.
    * On Mac/Linux, you can run `open index.html` or `xdg-open index.html` from the terminal.
